Chrome DevTools MCP Server

强大的开发工具与模型上下文协议的完美结合

🔧 什么是 Chrome DevTools？

Chrome DevTools 是 Google Chrome 浏览器内置的一套强大的开发者工具，为 Web 开发者提供了调试、性能分析、网络监控等全方位的开发支持。

通过 DevTools，开发者可以实时查看和修改 HTML/CSS、调试 JavaScript、分析网络请求、优化性能、模拟移动设备等，是现代 Web 开发不可或缺的工具。

🤖 什么是 MCP（Model Context Protocol）？

MCP（Model Context Protocol）是一个开放协议，旨在标准化 AI 模型与外部工具、数据源之间的交互方式。通过 MCP，AI 助手可以安全、高效地访问各种工具和服务。

MCP Server 充当 AI 模型和实际工具之间的桥梁，提供统一的接口和协议，使 AI 能够理解和使用各种开发工具。

⚡ Chrome DevTools MCP Server

Chrome DevTools MCP Server 将 Chrome DevTools 的强大功能通过 MCP 协议暴露给 AI 助手，使 AI 能够：

🌐 浏览器自动化

AI 可以控制浏览器导航、点击、填写表单等操作，实现智能化的 Web 交互

🔍 页面分析

获取页面快照、分析 DOM 结构、检查元素属性，帮助理解页面内容

🐛 调试辅助

查看控制台消息、网络请求、性能数据，协助开发者定位问题

📸 截图与录制

捕获页面截图、录制操作流程，便于文档和测试

⚙️ 设备模拟

模拟不同设备、网络条件、地理位置，测试响应式设计

🎨 样式调试

实时修改 CSS、测试样式效果，加速前端开发

💡 使用场景

Chrome DevTools MCP Server 可以应用于多种场景：

  • 智能测试：AI 辅助编写和执行 E2E 测试
  • 网页抓取：智能提取网页数据和内容
  • UI 调试：AI 协助定位和修复界面问题
  • 性能优化：自动分析性能瓶颈并提供建议
  • 可访问性检查：评估和改进网站的可访问性
  • 开发辅助：实时预览代码效果，加速开发流程

🚀 快速开始

要使用 Chrome DevTools MCP Server，你需要：

# 1. 安装 MCP Server
npm install -g chrome-devtools-mcp-server

# 2. 启动服务
chrome-devtools-mcp-server

# 3. 在 AI 助手中配置 MCP 连接
# 配置文件示例：mcp.json
{
  "mcpServers": {
    "chrome-devtools": {
      "command": "chrome-devtools-mcp-server",
      "args": []
    }
  }
}
